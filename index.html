Usage

  • Quick Gallery automatically looks for higher resolution images for all the thumbnails listed in the DIV element with ID of quickGalleryThumbs. To add more images to the gallery just insert more thumbnails in the list.
  • Thumbnails should be named the same as their high-resolution counterparts, except they should have "-thumb" appended to the filename.
  • Arrow keys can also be used to navigate between the photos while the modal is popped up.
